# Download Pop!_OS
###### from-dead-box-to-dev-box/00-overview/download-popos.md

Pop!_OS is a strong choice for young engineers because it is simple, reliable, and designed to stay out of the way. It provides a clean environment where the computer behaves predictably, without distractions, advertisements, or unnecessary software. Pop!_OS is also well‑supported, secure, and works well on older computers, giving them a fresh start and extending their useful life.

Pop!_OS also does **not** include any AI assistants or AI‑driven features by default. This keeps the environment focused, quiet, and free from automated suggestions. For young engineers, this creates a space where they can think independently, learn core skills, and build confidence without the computer trying to guide or influence their decisions.

This document guides a parent or guardian through downloading the Pop!_OS operating system. These steps are performed on a separate support machine that already works and has access to the Internet.

Follow each step in order.

---

## 1. Open a web browser

1. On the support machine, open any modern web browser (for example, Chrome, Edge, Firefox, or Safari).

---

## 2. Go to the official Pop!_OS download page

1. In the browser’s address bar, type:  
   **https://pop.system76.com**
1. Press Enter to load the official Pop!_OS website.
1. Locate the **Download** section on the page.

---

## 3. Choose the correct Pop!_OS version

1. Select the Pop!_OS version recommended for most computers (usually the latest LTS release).
1. If the target machine has an NVIDIA graphics card, choose the **NVIDIA** version.  
   If you are unsure, choose the standard version.

---

## 4. Download the ISO file

1. Click the download button for the selected version.
1. The browser will begin downloading a file ending in **.iso**.  
   This file is the complete Pop!_OS installer.

---

## 5. Record the checksum

1. On the download page, locate the **checksum** value (a long string of letters and numbers).
1. Copy or write down this checksum.  
   You will use it later to confirm the download is correct and not corrupted.

---

## 6. Wait for the download to finish

1. Allow the ISO file to finish downloading completely.  
   This may take several minutes depending on Internet speed.
