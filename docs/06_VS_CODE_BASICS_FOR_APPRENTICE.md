# VS Code Basics for the Apprentice Engineer  
###### <C:/dev/repos/dead-box-to-dev-box/docs/06_VS_CODE_BASICS_FOR_APPRENTICE.md>

This document teaches the apprentice how to operate Visual Studio Code (VS Code).  
These skills are required before writing multi‑file D programs.

Follow each step exactly.

---

# 1. Open VS Code

Press:

```
Super
```

Type:

```
Visual Studio Code
```

Press **ENTER**.

Operator rule:

- VS Code is always opened from the system launcher, never from the file browser.

---

# 2. Open a Folder

Press:

```
Ctrl + K, Ctrl + O
```

Navigate to the project folder.

Example:

```
/home/benny/projects/hello-world
```

Press **ENTER**.

Operator rule:

- The apprentice always opens the *folder*, not individual files.

---

# 3. Create a New File

Press:

```
Ctrl + N
```

Save it with:

```
Ctrl + S
```

Type the filename:

```
main.d
```

Press **ENTER**.

Operator rule:

- All source files must have the `.d` extension.

---

# 4. Save Files

Press:

```
Ctrl + S
```

Operator rule:

- The apprentice saves constantly.  
- No unsaved buffers before compiling.

---

# 5. Open the Integrated Terminal

Press:

```
Ctrl + `
```

This opens a Terminal inside VS Code.

Operator rule:

- The apprentice compiles from the integrated Terminal, not the external one, unless instructed.

---

# 6. Split the Editor

Press:

```
Ctrl + \
```

This splits the editor into two panes.

Operator rule:

- The apprentice uses splits only when needed.

---

# 7. Switch Between Tabs

Press:

```
Ctrl + Tab
```

Hold **Ctrl**, tap **Tab** to cycle.

Operator rule:

- The apprentice keeps the workspace clean and closes unused tabs.

---

# 8. Close a Tab

Press:

```
Ctrl + W
```

Operator rule:

- The apprentice closes files when done to reduce clutter.

---

# 9. Search Within a File

Press:

```
Ctrl + F
```

Type the search term.

Operator rule:

- The apprentice uses search instead of scrolling.

---

# 10. Search Across Files

Press:

```
Ctrl + Shift + F
```

Type the search term.

Operator rule:

- The apprentice uses global search to find definitions and references.

---

# 11. Command Palette

Press:

```
Ctrl + Shift + P
```

This opens the command palette.

Operator rule:

- The apprentice uses the palette only when instructed.

---

# 12. Completion

The apprentice is complete when they can:

- open VS Code  
- open a folder  
- create files  
- save files  
- use the integrated Terminal  
- split the editor  
- switch tabs  
- close tabs  
- search within files  
- search across files  

These skills are required for all future operator work.
