# Get Started With PopOS!
###### C:/dev/repos/from-dead-box-to-dev-box/guided-assembly/20-get-started-with-popos.md

This guide is used **after** the computer has been fully rebuilt and PopOS! is installed, updated, and ready for daily use. At this point, the young engineer has a clean, reliable system with no distractions and no unnecessary software. This document helps them become comfortable using PopOS! by connecting what they already know from Windows or macOS to the equivalent tools in PopOS!.

The goal is simple: help the young engineer feel at home right away.

---

## 1. The Global Search Box (like Windows Search or macOS Spotlight)

PopOS! includes a global search box that works the same way as the search tools on Windows and macOS.

**How to open it:**

1. Press the **Super** key.  
2. Start typing the name of anything you want to open:
   - apps  
   - settings  
   - files  
   - system tools  

This is the fastest way to move around PopOS!.

---

## 2. The File Manager (like Windows File Explorer or macOS Finder)

PopOS! includes a built‑in file manager called **Files**.

**How to open it:**

1. Press the **Super** key.  
2. Type **Files**.  
3. Press **Enter**.

**What it can do:**

- browse folders  
- open documents  
- move, copy, and rename files  
- connect to USB drives  
- show downloads and pictures  

The layout is simple and familiar.

---

## 3. The Terminal (like PowerShell, Command Prompt, or macOS Terminal)

PopOS! includes a terminal for advanced tasks. It is not required for everyday use, but it is helpful to know where it is.

**How to open it:**

1. Press the **Super** key.  
2. Type **Terminal**.  
3. Press **Enter**.

The terminal allows engineers to run commands, install tools, and automate tasks.

---

## 4. The Web Browser (like Chrome, Edge, Safari, or Firefox)

PopOS! includes a modern web browser by default.

**How to open it:**

1. Press the **Super** key.  
2. Type **Browser** or **Firefox**.  
3. Press **Enter**.

From here, the young engineer can:

- search the web  
- download files  
- watch videos  
- sign in to school accounts  
- access learning resources  

---

## 5. System Settings (like Windows Settings or macOS System Settings)

PopOS! has a clean, simple Settings app.

**How to open it:**

1. Press the **Super** key.  
2. Type **Settings**.  
3. Press **Enter**.

**Inside Settings, you can adjust:**

- Wi‑Fi  
- displays  
- sound  
- keyboard and mouse  
- updates  
- user accounts  

Everything is organized in a clear left‑side menu.

---

## 6. The Dock (like the macOS Dock or Windows Taskbar)

PopOS! includes a dock at the bottom of the screen.

**What it does:**

- shows open apps  
- lets you pin favorite apps  
- lets you switch between apps quickly  

This makes PopOS! feel familiar to both Windows and macOS users.

---

## 7. Workspaces (optional, like macOS Mission Control or Windows Virtual Desktops)

PopOS! supports workspaces, which are like extra desktops you can switch between.

**How to open the workspace view:**

1. Press the **Super** key.  
2. Click the workspace thumbnails on the right side.

This is optional but helpful for staying organized.

---

## Summary

PopOS! includes familiar tools that match what students already know:

- **Global Search** → like Windows Search or Spotlight  
- **Files** → like File Explorer or Finder  
- **Terminal** → like PowerShell, Command Prompt, or macOS Terminal  
- **Browser** → same as any modern browser  
- **Settings** → like Windows Settings or macOS System Settings  
- **Dock** → like the macOS Dock or Windows Taskbar  

With these basics, a young engineer can confidently begin using PopOS! for daily learning and exploration.
