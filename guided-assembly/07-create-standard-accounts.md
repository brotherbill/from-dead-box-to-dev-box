# Create Standard Accounts

This step creates the standard (non‑administrator) accounts that will be used on this computer.

Most households will create one standard account for the student. A parent standard account is optional and should only be created if the parent will use this computer for daily browsing, email, or personal work.

If the parent will not use this computer, do not create a parent standard account. The parent does not need a standard account to supervise the student.

**Note:** All Full Name and Username values shown in this document are examples. Replace them with the actual names you want to use. Usernames must be lowercase and contain no spaces.

---

## Optional: Create a Parent Daily‑Use Account

Create this account only if the parent will personally use this computer.

1. Press the **Super** key, type **Settings**, then press **Enter**.
1. Select **System & accounts → Users**.
1. Click **Add User**.
1. Enter example values (replace with your actual names):
   - **Full Name:** Best Parent  
   - **Username:** best-parent  
1. Set a password for the account.
1. Ensure **Administrator** is not selected.
1. Click **Add** to create the account.

If the parent will not use this computer, skip this section.

---

## Create the Student Account

1. In **Settings → Users**, click **Add User**.
1. Enter example values (replace with your actual names):
   - **Full Name:** Power Student  
   - **Username:** power-student  
1. Set a password for the student account.
1. Ensure **Administrator** is not selected.
1. Click **Add** to create the account.

---

## Sign Out and Verify Accounts

1. Click the **top‑right system menu**, then click the **right‑most Power On/Off symbol**.
1. Click **Sign out**.
1. When the confirmation dialog appears, click **Confirm**.
1. On the **Sign In** screen, confirm that you see the accounts you created.
1. Sign in to each account once to confirm that they open successfully.
